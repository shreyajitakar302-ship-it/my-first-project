# Quiz App

## About Project
Quiz App is an Android application developed using Java and Android Studio. The app allows users to answer multiple-choice questions and view their scores at the end of the quiz.

## Features
- Multiple Choice Questions
- Score Calculation
- User-Friendly Interface
- Instant Result Display

## Technologies Used
- Java
- Android Studio
- XML
